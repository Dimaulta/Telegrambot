# AntispamNowBot — технические заметки

- **Назначение сервиса**: антиспам бот с функциями выключателя на ночь, капчи для вступления и запрещателя сообщений от каналов.
- **Функции**:
  - `nightMode` — выключатель на ночь (автоматическое отключение уведомлений/сообщений).
  - `captcha` — капча для вступления в группу.
  - `channelBlock` — запрещатель сообщений от каналов.
- **Логирование**: минимум — тип действия, идентификатор пользователя/группы, результат операции.
- **Безопасность**: токен Telegram (`ANTISPAMNOWBOT_TOKEN`) хранить только в `config/.env`.
